# tayanon-primo-pre-interview-test

### วิธีการทำงาน

#### เริ่มจาก Install node_module

```
npm install
```

---

#### ทดสอบการทำงานของ Function โดย

```
npm run dev
```

หากอยากทดสอบ Input อื่นให้แก้ค่าที่อยากทดสอบจากใน Code ในไฟล์ src/merge.ts บรรทัดที่ 47-49

---

#### ทดสอบ Unit Test โดย

```
npm run test
```
